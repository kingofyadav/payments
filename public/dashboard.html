Browser Checkout SDK

            
<script src="/v1/checkout.js"></script>
<script>
const pe = new PayEngine({ key: 'key_id' });

pe.open({
  order_id:    'order_xyz',
  amount:      50000,
  currency:    'INR',
  name:        'Acme Store',
  description: 'Order #1234',
  prefill: { name: 'Priya Sharma', email: 'priya@example.com' },
  handler: function(response) {
    // Send response.razorpay_payment_id to your server for verification
    console.log('Payment ID:', response.razorpay_payment_id);
  },
  modal: { ondismiss: function() { console.log('Checkout closed'); } },
});
</script>

          


          

            
Integration Guide

            

              
1
Create an order
POST /v1/orders with amount in paise. Returns order_id. Send X-Idempotency-Key to safely retry.

              
2
Open checkout
Load /v1/checkout.js and call pe.open({ order_id, amount, handler }). Customer picks UPI / Card / Net Banking.

              
3
Verify on server
Your handler receives the payment ID. Fetch GET /v1/payments/:id on your server to confirm status: captured.

              
4
Register a webhook
POST /v1/webhooks with your endpoint URL. Receive payment.captured events. Always verify X-Gateway-Signature.

              
5
Issue refunds
POST /v1/refunds with payment_id and optional partial amount. Settlements are processed T+2.
